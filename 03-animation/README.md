# Day Three - Animation

## What You Have

You currently have a series of files: the normal `index.html`, index.js, and main.css you would expect. Additionally, you have `gsap.min.js` which includes `gsap` code.

The `finished-example` files are code that constructs a working version of the animation. Open `finished-example.html` in your browser to see what the completed product should look like.

## Deliverable

Build out the animation sequence modeled in `finished-example.html`. You should reference the `finished-example.js` and `finished-example.css` only when stuck.
